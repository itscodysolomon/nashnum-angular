What is Nashnum?

Nashnum is a simple tool made to help musicians become more familiarized with the Nashville Number System.

The Nashville Number System is a method of transcribing music by denoting the scale degree on which a chord is built.

For example, in the key of C, it would be:C = 1, Dm = 2, Em = 3, F = 4, G = 5, Am = 6, BÂº = 7
